Technological Infraestructure - #back to top

System must have 3 environments: development, staging, production

Development: Environment for create new features without any risk of broke something that can affect a real user.

Staging: Environment for test finished features and test integration with real data, the purpose is to ensure the quality of the system

Production: Environment for real use and with real users.

Necessary resources.

  • 2 server with 2 CPU and 4 GB memory
  • 2 server with 4 CPU and 8 GB memory
  • 2 S3 bucket instance
140$ USD month (Estimate)

*For development stage we only need 1 server with 2 CPU and 4 GB memory
